Create a markdown editor with live preview. Create these files in the current directory:
1. index.html - Split-pane layout: left side is a textarea editor, right side is live preview. Include a toolbar with buttons for Bold, Italic, Heading (H1-H3), Link, Image, Code block, Unordered list, Ordered list, Blockquote, Horizontal rule. Add an "Export HTML" button that downloads the rendered output.
2. style.css - Professional dark theme, responsive split panes with draggable resizer, syntax-highlighted toolbar, smooth transitions, styled preview matching GitHub markdown style.
3. markdown-parser.js - A complete markdown parser (no external libraries) that handles: headings (h1-h6), bold, italic, bold+italic, inline code, code blocks with language labels, links, images, unordered and ordered lists (including nested), blockquotes (including nested), horizontal rules, paragraphs, line breaks, and escape characters.
4. editor.js - Editor logic: real-time preview update on input, toolbar button actions that insert markdown syntax at cursor position (wrapping selected text if any), drag-to-resize split panes, keyboard shortcuts (Ctrl+B bold, Ctrl+I italic, Ctrl+K link), localStorage auto-save/restore, and Export HTML functionality.
